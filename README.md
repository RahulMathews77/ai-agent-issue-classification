# AI Agent for Issue Classification & Root Cause Analysis

## Overview
This project is a modular AI agent that processes unstructured issue descriptions and classifies them using semantic retrieval and LLM-based reasoning.

## Features
- Semantic search using Sentence Transformers
- Tool-based modular architecture
- Local LLM integration using Ollama
- Confidence-based fallback system
- Human-in-the-loop feedback

## Tech Stack
Python, Sentence Transformers, Scikit-learn, Ollama, JSON

## How to Run
1. Install dependencies
2. Start Ollama:
   ollama serve
3. Run:
   python3 main.py
